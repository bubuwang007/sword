# SWord

## 项目概述

提供一个简洁、易用的 Python 封装库，用于读写 Microsoft Word 文档（`.docx` 格式）。

## 技术栈

- **Python**: 3.11+
- **核心依赖**: `python-docx`
- **类型检查**: `mypy`
- **代码格式化**: `black`
- **单元测试**: `pytest`

## 编码规范

- 遵循 PEP 8
- 使用类型注解（type hints）
- 所有公开 API 需要文档字符串
- 函数单一职责原则

## 计划和任务

制定的计划和任务写入`.claude/docs`下的`plan`和`task`文件夹中，分类根据项目文件夹结构进行组织。

## 测试

- 编写单元测试覆盖核心功能
- 使用 `pytest` 进行测试运行
- 所有测试文件命名以 `test_` 开头
- 测试文件放在项目根目录下的 `tests` 文件夹中，根据项目文件夹结构进行组织
